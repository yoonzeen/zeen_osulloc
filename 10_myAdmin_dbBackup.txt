++++++++++++ db 백업하기 +++++++++++++++++++

1) xampp > mysql 오른쪽에 Admin 버튼 클릭
2) phpMyAdmin 접속 페이지 확인

3) 화면 오른쪽 상단에 데이터베이스 탭 클릭
: 현재 보관 중인 데이터베이스들 확인 가능

4) db1을 클릭하면 db1에 담고 있는 테이블들
확인 가능 !

5) 사용자가 보관해야 하는 테이블들 선택하기
(체크박스 체크), 다중선택 가능

6) 상단의 탭들 중에 내보내기 탭 클릭
7) 형식들 중에서 sql 선택하고, 바로 하단의 실행 버튼 클릭

+++++++++ db 가져오기 ++++++++++++++++
1) db1을 사용하고 싶으나 같은 이름을 만들지 못하면 다른
이름으로 db명 만들기
	ex) dbs1
2) 오른쪽 상단의 데이터베이스 탭 클릭
: 아래 데이터베이스 명에 dbs1 기입
: 바로 오른쪽 데이터 정렬 방식에서 문자열 셋팅 (utf8_general_ci 선택)
: 만들기 버튼 클릭

3) dbs1에서 오른쪽 상단의 가져오기 탭 클릭 > 파일 선택 버튼에서 백업 
받은 자료 선택 후 가장 하단의 실행 버튼 클릭
